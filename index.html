프로젝트 소개

테마파크 월드투어 웹사이트 - 관광자원론 적용 사례

프로젝트 목적

본 웹사이트는 관광자원론 수업의 일환으로 제작된 테마파크 관광 상품 안내 플랫폼입니다. 전 세계 주요 테마파크를 대상으로 관광자원의 특성, 경제적 파급효과, 그리고 관광 산업의 트렌드를 분석하고 시각화하여 제공합니다.

관광자원론 이론 적용

관광자원 분류

테마파크를 인공 관광자원으로 분류하고, 판타지/영화/모험 등 6가지 테마로 세분화하여 분석했습니다.

관광 매력성

각 테마파크의 독창성, 접근성, 편의성을 평가하고 TOP 10 순위를 통해 글로벌 경쟁력을 비교했습니다.

경제적 파급효과

테마파크 산업이 창출하는 직·간접 수익($147B)과 일자리(350만 개)를 정량적으로 분석했습니다.

지리적 분포

15개국 475개 테마파크의 공간적 분포를 버블 맵으로 시각화하여 지역별 발달 양상을 파악했습니다.

시간적 변화

COVID-19 팬데믹 전후 방문객 추이를 분석하여 관광 산업의 회복 탄력성을 검증했습니다.

주요 기능

  • 투어 루트 시스템 - 멀티 국가 패키지 상품
  • 상세 일정표 - Day별 활동, 식사, 숙박 정보
  • 실시간 검색 - 테마파크/투어 통합 검색
  • 비용 계산기 - 인원/옵션별 맞춤 견적
  • 통계 대시보드 - 7가지 인터랙티브 차트
  • API 연동 - 날씨, 환율, 파크 정보

기술 스택

  • Frontend HTML5, CSS3, JavaScript (Vanilla)
  • API OpenWeather, ExchangeRate, ThemeParks Wiki
  • Data Viz SVG Charts, CSS Animations
  • Security XSS Prevention (escapeHtml)
  • Storage LocalStorage, Cache API
